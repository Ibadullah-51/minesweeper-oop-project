# Minesweeper Game (Python OOP)

## Project Description

This is a **Minesweeper game** developed in **Python** using **Object-Oriented Programming (OOP)** concepts.
The game provides a graphical interface where players can uncover cells while avoiding hidden mines.

---

## Features

* Fully functional Minesweeper game
* Implemented using OOP concepts (Classes & Objects)
* Random mine generation
* Cell number hints
* Click-based interaction
* Simple GUI (Tkinter / PyGame – edit based on your project)

---

## Technologies Used

* Python
